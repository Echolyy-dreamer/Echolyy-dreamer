# AWS Solutions Architect  — Top 10 架构题

包括典型场景、考察点、关键回答话术，便于面试前复习与展示。

---

## 1️⃣ 全球电商平台架构设计
**场景**：设计一个全球电商系统，支持高并发、低延迟、跨区域访问。  

**考点**：
- 跨 Region / AZ 高可用性
- CDN + WAF
- 弹性伸缩（Auto Scaling + ALB）
- 数据分层：RDS / DynamoDB / ElastiCache
- 异步任务处理（SQS / EventBridge）

**面试话术**：
- 先讲需求，再分析痛点
- 分层架构、负载均衡、缓存策略
- RPO/RTO、灾难恢复、一致性策略

---

## 2️⃣ 全球金融交易系统架构
**场景**：London ↔ Singapore 交易清算平台，低延迟 (<100ms)，强一致性，极端容灾。  

**考点**：
- AWS Global Accelerator 优化网络
- Aurora Global Database / 单源写、跨区域只读
- Transit Gateway Peering
- 本地读、远程写策略
- 灾难恢复 RPO=0、RTO<5min

**面试话术**：
- 强调 Speed-of-Light 限制
- 同步/异步策略权衡
- 数据最终一致性 vs 强一致性
- 架构图展示跨 Region 同步

---

## 3️⃣ 小型公司 LAMP → 云原生迁移
**场景**：单机 LAMP → AWS，实现弹性、高可用、低成本、易运维。  

**考点**：
- ALB + Auto Scaling + EC2 / 容器
- RDS Multi-AZ + Read Replica
- S3 + ElastiCache 缓存
- IAM + IaC
- 灾难恢复 & 多环境管理

**面试话术**：
- 自然增长与非线性扩展
- 数据安全、长期存储策略
- 自修复能力（Auto Scaling、Health Check）

---

## 4️⃣ 异步事件驱动架构设计
**场景**：每秒 10k+ 日志或交易数据处理流水线，需实时分析、报警、入湖。  

**考点**：
- Kinesis / SQS / EventBridge 消息流
- Lambda / Fargate 异步处理
- Fan-out / Backpressure / 幂等设计
- 最终一致性、可恢复性
- 数据湖写入（S3 + Athena / Glue）

**面试话术**：
- 松耦合、无阻塞设计
- 高吞吐、流量波动处理
- 灾难恢复和多 AZ / Region 容错

---

## 5️⃣ 缓存与数据库一致性
**场景**：Redis + RDS，缓存命中率高但需强一致。  

**考点**：
- 缓存策略：Write-Through / Write-Back / Cache-Aside
- 分布式锁（DynamoDB Lock / Redlock）
- 数据持久化（RDS WAL / Redis AOF / RDB）
- 异步更新策略与最终一致性

**面试话术**：
- 强一致 vs 最终一致性分析
- 崩溃恢复流程、分布式锁使用场景

---

## 6️⃣ 高可用 Auto Scaling 流程
**场景**：EC2 + Auto Scaling + CloudWatch 自适应扩展。  

**考点**：
- CloudWatch Alarm 触发
- Auto Scaling Group 扩缩容
- ELB / ALB 负载均衡分配
- 健康检查 & 自修复
- Least Outstanding Requests 调度逻辑

**面试话术**：
- 描述触发条件、冷却时间、跨 AZ 扩展
- 强调稳定性与可观察性

---

## 7️⃣ ELB / ALB / NLB 跨 AZ 高可用性
**场景**：解释负载均衡器在多 AZ 下的行为。  

**考点**：
- 跨 AZ 流量分发
- 健康检查与剔除不健康实例
- Least Outstanding Requests、Round-Robin
- 容错机制

**面试话术**：
- 跨 AZ 高可用性
- 调度算法对延迟/吞吐影响

---

## 8️⃣ RDS 多 AZ + 备份策略
**场景**：Multi-AZ RDS 高可用与持久化。  

**考点**：
- Standby 不对外提供服务
- 自动故障切换
- Snapshot / PITR / 自动备份
- RPO / RTO

**面试话术**：
- 备份 ≠ Multi-AZ 替代
- 持久化与快照区别

---

## 9️⃣ 跨国访问加速与优化
**场景**：海外用户访问国内服务或国内用户访问海外服务。  

**考点**：
- AWS Global Accelerator / Route 53 Latency-Based Routing
- CloudFront CDN + Lambda@Edge / API Gateway
- Direct Connect / VPN / Transit Gateway
- 网络分层优化策略

**面试话术**：
- 低延迟 + 高可靠
- 动态/静态内容不同策略

---
